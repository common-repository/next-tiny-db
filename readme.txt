Next Tiny DB allows you to benefit from a small database on any of your web pages through shortcodes.
Your visitors can then search and display one or more results according to their search criteria.
Configure the layout of your different fields per column.
Possibility to display images according to each records, and text or html file content instead of any empty field.
It simply works from any Excel .CSV file.
You can manage several databases and have as many shortcodes you need on your web pages to satisfy your visitors!

== Description ==
Next Tiny DB allows you to benefit from a small database on any of your web pages through shortcodes.
Your visitors can then search and display one or more results according to their search criteria.
Configure the layout of your different fields per column.
Possibility to display images according to each records, and text or html file content instead of any empty field.
It simply works from any Excel .CSV file.
You can manage several databases and have as many shortcodes you need on your web pages to satisfy your visitors!

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/next-tiny-db` directory, or install the plugin through the WordPress plugin screen directly.
3. Activate the plugin through the 'Plugins' screen in WordPress
2. Go to Next Tiny DB menu to configure the plugin and save your changes.
